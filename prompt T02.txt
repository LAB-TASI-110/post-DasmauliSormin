TUJUAN (Apa yang ingin dicapai)

Buat program yang dapat menghitung **total stok barang berdasarkan kode kategori yang diminta** dari sekumpulan data stok gudang.

Program harus mampu:

* Menerima banyak data stok
* Memisahkan berdasarkan kategori
* Menjumlahkan stok hanya untuk kategori tertentu
* Menghasilkan output total stok kategori tersebut

## 2️⃣ MASALAH UTAMA

Gudang memiliki banyak barang dengan kategori berbeda.
Admin gudang hanya ingin mengetahui **total stok dari satu kategori tertentu** tanpa menghitung semua kategori.

Masalah:

* Data stok bercampur antar kategori
* Perlu filtering berdasarkan kode kategori
* Perlu perhitungan total stok kategori target

## 3️⃣ ALUR SOLUSI

1. Input jumlah data barang (N)
2. Input data barang (kategori + jumlah stok)
3. Input kategori yang ingin dihitung
4. Filter data sesuai kategori
5. Jumlahkan stok kategori tersebut
6. Tampilkan total stok

## 4️⃣ DESKRIPSI OBJEKTIF SISTEM

Sistem adalah program CLI (terminal) yang:

* Mengolah input numerik & string
* Menggunakan struktur data list
* Melakukan filtering & agregasi data
* Menghasilkan satu nilai output numerik

---

# ⚙️ SPESIFIKASI TEKNIS

## Library

Hanya standard library Python (tanpa library eksternal)

## Database / Sistem Manajemen

Tidak menggunakan database (in-memory processing)

# 🏗️ STRUKTUR PROGRAM

## Struktur Data

Gunakan list of dictionary:

```
barang = [
   {"kategori": "GULA", "stok": 85},
   {"kategori": "MINYAK", "stok": 120}
]
```

---

## INPUT PROGRAM

1. N → jumlah data barang
2. N baris data:

   * kode kategori (string)
   * jumlah stok (integer)
3. kategori target (string)

Contoh input:

```
4
GULA 85
MINYAK 120
SERAGAM 5
BUKU 22
GULA
```

---

## OUTPUT PROGRAM

Output:

```
Total stok kategori GULA = 85
```

Jika tidak ditemukan:

```
Kategori tidak ditemukan
```
